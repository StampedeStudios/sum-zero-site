# Sum Zero

Website for [Sum Zero](https://github.com/StampedeStudios/sum-zero), a game by Stampede Studios.

## Development

Run ReScript in dev mode:

```sh
pnpm res:dev
```

Start up dev server:

```sh
pnpm dev
```

## Build & Preview

```sh
pnpm res:build
pnpm build
pnpm preview
```

Output is in the `dist` folder.
